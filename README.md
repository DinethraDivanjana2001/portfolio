## üöÄ Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/DinethraDivanjana2001/minimal-next-portfolio.git
   cd minimal-next-portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Create `.env.local` file (optional for contact form):
   ```env
   GOOGLE_FORM_LINK=your_google_form_link
   GOOGLE_FORM_FIELD_ID_NAME=entry.xxxxx
   GOOGLE_FORM_FIELD_ID_EMAIL=entry.xxxxx
   GOOGLE_FORM_FIELD_ID_MESSAGE=entry.xxxxx
   GOOGLE_FORM_FIELD_ID_SOCIAL=entry.xxxxx
   ```

4. Run development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000)
